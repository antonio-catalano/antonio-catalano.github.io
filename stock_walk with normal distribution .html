''' Simulazione del prezzo di un'azione seguendo il modello Black-Scholes su un arco temporale di 3 anni '''

stock_price = 100 # prezzo iniziale dell'azione
u = 0.25  # rendimento atteso annuo
sigma = 0.3 # volatilita' annua
_Paths = []
for i in range(756):
        dx = stock_price * ( u*(1/252) + sigma*np.sqrt(1/252)*np.random.randn() )
        stock_price += dx
        _Paths.append(stock_price)
        
_Paths = pd.Series(_Paths)   

def logR(lista):
        i = 0
        Rend = []
        while i <= len(lista)-2:
            Rend.append(np.log(lista[i+1]/lista[i]))
            i += 1
        return Rend
    
logR_daily = np.array(logR(_Paths))

def maxVola(array):
    if array.max() >= abs(array.min()):
        return array.max()
    else:
        return array.min()
        
max_vola = maxVola(logR_daily)
max_range_oscillazione = max_vola / logR_daily.std()

print("Daily standard deviation:", logR_daily.std())
print("Massima volatilita' registrata:", max_vola)
print("Massima escursione di rendimento giornaliero rispetto alla standard deviation giornaliera:", max_range_oscillazione )

fig, (ax1,ax2) = plt.subplots(2,1, figsize=(10,15))
ax1.plot(logR_daily, label= 'log return daily')
ax2.plot(_Paths,'r--', label= 'stock price')
ax1.set_xlabel('trading day')
ax2.set_xlabel('trading day')
ax1.legend();
ax2.legend();
    

Daily standard deviation: 0.01933538948155934
Massima volatilita' registrata: 0.06972988432289755
Massima escursione di rendimento giornaliero rispetto alla standard deviation giornaliera: 3.606334611950835

In questa simulazione, su 3 anni di trading il rendimento daily non ha mai superato in valore assoluto piu' di 3,6 volte la propria deviazione standard giornaliera.
Inoltre i rendimenti daily sono distribuiti con grande frequenza intorno a valori medio-distanti dal rendimento giornaliero medio e con probabilita' quasi nulla oltrepassano le 4 deviazioni standard in up o in down
La realt diversa:
il prezzo di uno stock si muove con altissima frequenza con rendimenti vicini a quello atteso giornaliero, con bassa frequenza intorno a quelli medio-distanti, e con piu' alta frequenza rispetto al modello di black-scholes fa picchi in up o in down che possono arrivare anche a 10-15 volte la deviazione standard giornaliera stimata.
